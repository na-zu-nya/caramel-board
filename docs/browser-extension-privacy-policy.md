# Caramel Board Clipper BETA プライバシーポリシー

最終更新日: 2026-06-20

Caramel Board Clipper BETA は、Webページ上でユーザーが右クリックした画像または直URL動画を、ユーザー自身が設定した Caramel Board API へ追加するための拡張です。

## 収集・保存する情報

拡張は、機能提供のために次の情報を扱います。

- ユーザーが設定した Caramel Board API URL
- Caramel Board で発行した Clipper連携キー
- ユーザーが任意で設定した Basic Auth のユーザー名とパスワード
- Caramel Board から取得したライブラリ一覧、ライブラリ色、保護状態、認証状態
- ユーザーが右クリックして追加した画像または直URL動画
- 追加対象のURL、ファイル名、Content-Type など、取り込みに必要なメタデータ
- 保護付きライブラリを認証するために入力されたパスワード

API URL、Clipper連携キー、Basic Auth 情報、ライブラリ一覧は Chrome のローカル拡張ストレージに保存されます。
保護付きライブラリのパスワードは認証リクエストに使用されますが、拡張ストレージには保存しません。

## 利用目的

これらの情報は、次の目的にだけ使用します。

- 選択された画像または直URL動画を Caramel Board ライブラリへ追加する
- ライブラリ一覧を右クリックメニューに表示する
- 保護付きライブラリを認証する
- 追加結果をページ内の一時的なトーストで表示する
- Caramel Board API との接続を確認する

## 送信先

拡張が送信するデータは、ユーザーが設定した Caramel Board API URL 宛てです。
開発者のサーバ、広告ネットワーク、分析サービス、その他の第三者サービスへデータを送信しません。

対象サイトから画像または直URL動画を取得するため、ユーザーが右クリックしたメディアURLへリクエストを行う場合があります。
この取得はユーザー操作に基づく追加処理のためだけに行います。

## 第三者提供

開発者は、この拡張が扱う情報を販売、貸与、広告目的の共有、分析目的の共有に利用しません。
法令上必要な場合を除き、第三者へ提供しません。

## 保持と削除

Chrome のローカル拡張ストレージに保存された設定は、ユーザーが拡張の設定画面で上書きするか、Chrome から拡張を削除するまで保持されます。
Caramel Board に追加されたファイルやメタデータは、ユーザー自身の Caramel Board 環境の保存設定に従って保持されます。

## セキュリティ

Clipper連携キーと Basic Auth 情報は Chrome のローカル拡張ストレージに保存されます。
端末やChromeプロファイルを共有している場合は、OSアカウントやChromeプロファイルの保護を行ってください。
Caramel Board API をインターネットへ直接公開する運用は想定していません。

## 対象外

YouTube、niconico などのストリーミングプレイヤー、HLS / DASH、DRM付き動画、`blob:` URL は取り込み対象外です。
BETA 版のため、仕様やデータの扱いは今後変更される可能性があります。
重要な変更がある場合は、このポリシーを更新します。
